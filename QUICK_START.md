# تعليمات التشغيل السريع

## خطوات التشغيل

### 1. تثبيت التبعيات
```bash
npm install
```

### 2. إعداد متغيرات البيئة
```bash
# انسخ ملف البيئة
cp env.example .env.local

# قم بتعديل القيم في ملف .env.local
```

### 3. إعداد قاعدة البيانات
```bash
# توليد Prisma Client
npx prisma generate

# إنشاء قاعدة البيانات وتطبيق المخطط
npx prisma db push

# إدراج البيانات التجريبية
npx prisma db seed
```

### 4. تشغيل المشروع
```bash
# وضع التطوير
npm run dev

# البناء للإنتاج
npm run build

# تشغيل الإنتاج
npm run start
```
